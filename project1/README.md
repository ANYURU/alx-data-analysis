# Introduction
In this project I investigated the No show appointments dataset. 

# Instructions
1. Install [Anaconda](https://www.anaconda.com/products/distribution), conda and jupyter notebook on your machine 
2. Clone the repository.
2. Create a virtual environment.
```python
    python3 -m venv env
    source env/bin/activate
``` 
3. Run the `requirements.txt` file to install the requirements in your environment.
```python
    pip install -r requirements.txt
```
